# Argentina programa: #YoProgramo. Proyecto final

Aplicación web full stack que muestra: datos personales, estudios cursados, experiencia laboral, conocimiento de tecnologías y proyectos. Aplicación de arquitectura distribuida: diseño de interfaz para mostrar la información, una base de datos y APIs para proveer la información a través de internet. Tecnologías utilizadas: Angular + Java + MySQL.
